# AI Voice Assistant on Raspberry Pi 4

Offline-capable voice assistant system built on Raspberry Pi 4.

## Features
- Speech-to-text (Vosk offline)
- LLM (OpenAI API)
- Text-to-speech (espeak-ng)
- OLED-based state visualization
- Systemd autostart

./start.sh
